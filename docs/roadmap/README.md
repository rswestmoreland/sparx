# Roadmap and Phase Archive

This directory contains phase notes, checkpoint reviews, validation fix notes,
benchmark follow-ups, and handoff prompts. These files are retained for project
traceability and should not be treated as the active user-facing documentation
set.

Use the active guides in the parent `docs/` directory for current architecture,
operations, alerting, configuration, validation, benchmarking, and release
readiness information.
